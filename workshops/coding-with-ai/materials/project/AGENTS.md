# Project Instructions

This is a workshop project for practicing AI-assisted coding.

When changing this project:

- keep changes small and focused
- inspect the code before editing
- run `npm test` after changing JavaScript logic
- use the browser for UI checks
- review `git diff` before committing
- do not add private data, API keys, or real user data

Prefer simple browser JavaScript, HTML, and CSS. Do not add a build step unless the facilitator asks for it.
