## How It Works:

1. **Timestamp Extraction**:
   ```javascript
   const time = new Date(data.timestamp || Date.now()).toLocaleTimeString();
   ```
   - If the server includes a `timestamp` field in the JSON, it uses that
   - If no timestamp is provided, it uses `Date.now()` (current time when message is received)
   - Converts to local time format (e.g., "2:30:45 PM")

2. **Timestamp Display**:
   The timestamp is wrapped in `[]` and prepended to every message type:
   - `[2:30:45 PM] Alice: Hello!`
   - `[2:31:00 PM] Bob joined`

## Complete Updated `onmessage` Handler:

```javascript
socket.onmessage = (event) => {
  try {
    const data = JSON.parse(event.data);
    const time = new Date(data.timestamp || Date.now()).toLocaleTimeString();
    
    switch (data.type) {
      case 'message':
        addMessage(`[${time}] ${data.name}: ${data.text}`);
        break;
      case 'join':
        addMessage(`[${time}] ${data.name} joined`);
        break;
      case 'leave':
        addMessage(`[${time}] ${data.name} left`);
        break;
      case 'nick':
        addMessage(`[${time}] ${data.oldName} changed name to ${data.name}`);
        break;
      case 'list':
        // Show in separate area instead of chat
        updateUserList(data.names);
        break;
      case 'error':
        addMessage(`[${time}] [ERROR] ${data.message}`);
        break;
      default:
        console.log('Unknown message type:', data.type);
    }
  } catch (e) {
    const errorTime = new Date().toLocaleTimeString();
    addMessage(`[${errorTime}] Invalid message received`);
  }
};
```
