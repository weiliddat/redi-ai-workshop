### 4. Create separate HTML Elements in JavaScript
Instead of adding one long string of text, create a `span` for the time and a `span` for the content. This allows for better styling and prevents "text injection" issues.

**Update your `addMessage` function like this:**

```javascript

  const div = document.createElement('div');
  div.className = 'message';

  // 1. Create the timestamp span
  const timeSpan = document.createElement('span');
  timeSpan.className = 'message-time';
  timeSpan.textContent = `[${time}]`;

  // 2. Create the content span
  const contentSpan = document.createElement('span');
  contentSpan.className = 'message-text';
  contentSpan.textContent = text;

  // 3. Append both to the main message div

  div.appendChild(contentSpan);
```
