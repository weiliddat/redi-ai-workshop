### Step 2: Update JavaScript to handle the command
We need to update the script to listen for the button click, send the JSON command to the server, and handle the response (updating the local state if successful).

Add the following code inside your `<script>` tag:

**1. Select the new elements:**
Add these lines near your existing variable declarations (after `messageInput`):
```javascript
const nickInput = document.getElementById('nickInput');
const nickBtn = document.getElementById('nickBtn');
```

**2. Add the nickname change handler:**
Add this function near your `connect` and `disconnect` functions:
```javascript
function changeNickname() {
  // Ensure we are connected
  if (!socket || socket.readyState !== WebSocket.OPEN) {
    alert('Not connected');
    return;
  }

  const newName = nickInput.value.trim();
  if (!newName) return;

  // Send the command to the server
  socket.send(JSON.stringify({
    type: 'nick',
    name: newName
  }));

  // Optional: Clear the input immediately
  nickInput.value = '';
}
```

**3. Add the event listener:**
Link the button to the function:
```javascript
nickBtn.addEventListener('click', changeNickname);
```

**4. Update the `onmessage` handler:**
You need to handle the server's response to the nickname change. Specifically, you need to update your local `nameInput` value if the change was successful (so future messages use the new name).

Find your existing `socket.onmessage` function and update the `else if` block for `data.type === 'nick'`:

```javascript
socket.onmessage = (event) => {
  try {
    const data = JSON.parse(event.data);
    const time = new Date(data.timestamp || Date.now()).toLocaleTimeString();

    if (data.type === 'message') {
      addMessage(`[${time}] ${data.name}: ${data.text}`);
    } else if (data.type === 'join') {
      addMessage(`[${time}] ${data.name} joined`);
    } else if (data.type === 'leave') {
      addMessage(`[${time}] ${data.name} left`);
    
    // --- NEW HANDLER FOR NICKNAME CHANGES ---
    } else if (data.type === 'nick') {
      // If this is the confirmation of *our* name change, update the local input
      if (data.oldName === nameInput.value) {
        nameInput.value = data.name;
      }
      addMessage(`[${time}] ${data.oldName} changed name to ${data.name}`);
    
    // -----------------------------------------

    } else if (data.type === 'error') {
      addMessage(`[ERROR] ${data.message}`);
    }
  } catch (e) {
    addMessage('Invalid message received');
  }
};
```

### Summary of Changes

1.  **HTML:** Added a new input (`#nickInput`) and button (`#nickBtn`) specifically for renaming.
2.  **JS - Send:** When the button is clicked, we send a JSON object `{ "type": "nick", "name": "..." }` to the server.
3.  **JS - Receive:** When the server broadcasts the name change (or sends a confirmation), we check if the `oldName` matches our current name. If it does, we update our local `nameInput` value so that future messages we send are labeled with the new name.
