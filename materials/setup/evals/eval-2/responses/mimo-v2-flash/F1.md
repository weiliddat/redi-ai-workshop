I'll show you how to modify your chat client to display timestamps next to each message. Here are the steps with explanations:

## Step 1: Modify the `addMessage` function to accept a timestamp parameter

```javascript
function addMessage(text, timestamp) {
  const div = document.createElement('div');
  div.className = 'message';
  
  // Format the timestamp if provided
  let timeStr = '';
  if (timestamp) {
    const time = new Date(timestamp).toLocaleTimeString();
    timeStr = `[${time}] `;
  }
  
  div.textContent = timeStr + text;
  messagesEl.appendChild(div);
  messagesEl.scrollTop = messagesEl.scrollHeight;
}
```

**Explanation:** 
- We're adding a `timestamp` parameter to the function
- If a timestamp is provided, we format it using `toLocaleTimeString()` for a readable format like "2:30:45 PM"
- We prepend the formatted timestamp in square brackets before the message text

## Step 2: Update the message handlers in the WebSocket `onmessage` event

Replace the existing `onmessage` handler with this updated version:

```javascript
socket.onmessage = (event) => {
  try {
    const data = JSON.parse(event.data);

    // Extract and format timestamp for consistency
    const timestamp = data.timestamp || Date.now();
    const timeStr = new Date(timestamp).toLocaleTimeString();

    if (data.type === 'message') {
      addMessage(`${data.name}: ${data.text}`, timestamp);
    } else if (data.type === 'join') {
      addMessage(`${data.name} joined`, timestamp);
    } else if (data.type === 'leave') {
      addMessage(`${data.name} left`, timestamp);
    } else if (data.type === 'error') {
      addMessage(`[ERROR] ${data.message}`, timestamp);
    }
  } catch (e) {
    addMessage('Invalid message received', Date.now());
  }
};
```

**Explanation:**
- We extract the timestamp from the server data or use current time if not provided
- We format it once and pass it to `addMessage()` for each message type
- The timestamp is now included in every message display
