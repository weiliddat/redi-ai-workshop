# Coding with AI — Slides

Slidev deck for the 60-minute Coding with AI workshop.

To present:

- `pnpm install`
- `pnpm dev`
- visit <http://localhost:3030>

The deck is intentionally thin. The [teacher runbook](../materials/teacher-runbook.md) is the facilitator script, and the [participant exercise brief](../materials/participant-exercise-brief.md) carries the hands-on details. Slides only prompt what the room needs to see at each moment.

Before presenting, replace the server URL placeholder on the practice slide with the real facilitator server URL.
