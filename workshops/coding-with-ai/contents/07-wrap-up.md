# 07 - Wrap-up

## Goal

End with one habit participants can use the next time they code with AI.

## Closing Questions

Point participants to the [AI coding checklist](../materials/ai-coding-checklist.md), then close with three questions:

- What is the smallest useful change?
- How will I check it?
- Can I explain the diff before I commit?

## Feedback Question

Before participants leave, ask one question, answered out loud or in Slack:

> Which step of the loop will you actually use next week?

The answers tell you which habit landed and what to emphasize next run.

## Final Message

AI is useful when it helps you ship better work and become a stronger developer.

You do not need to remember every prompt from today. Remember the loop:

clean start, inspect, plan, change, check, review the diff, commit, and record one lesson.
