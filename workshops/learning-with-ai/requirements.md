# Requirements

## Coordinator

- [ ] Access to at least one primary AI chat tool: ChatGPT, Gemini, or Claude
- [ ] OpenRouter account available as a backup option
- [ ] Workshop comms ready with the validated tool list and setup links

## Teacher

- [ ] Contents
- [ ] WebSocket chat server running and accessible to students (share the `wss://` URL at the start of section 02)
- [ ] API spec ready to paste into Zoom chat
- [ ] Access to a backup OpenRouter chat with `Step 3.5 Flash`

## Students

- [ ] Access to at least one primary AI chat tool: ChatGPT, Gemini, or Claude
- [ ] Optional: OpenRouter account set up as a backup, using `Step 3.5 Flash`
- [ ] A modern browser (Chrome, Firefox, Edge) that can open local HTML files
- [ ] Know how to open the browser developer console (F12 or Ctrl+Shift+I)
