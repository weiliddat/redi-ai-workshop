# REDI AI Workshops

This repo contains AI workshop material for REDI students and broader learner audiences.

There are now two workshop tracks:

1. **Learning with AI**  
   For students who are still learning to program and need to use AI without losing the skills they are building.

2. **Coding with AI**  
   For students, graduates, and external learners who use AI to code on projects.

## Workshop Tracks

```
workshops/
  learning-with-ai/       - Original workshop for students learning programming
  coding-with-ai/         - New workshop on coding with AI in project work
```

## Shared Principle

AI should make participants stronger over time.

For early learners, that means staying cognitively engaged while learning new skills.

For people coding with AI, that means using AI in a professional workflow:

- make small version-controlled changes
- understand the code you ship
- build feedback loops with tests, browser checks, and review
- improve the instructions and documentation that guide future work

## Current Status

`workshops/learning-with-ai/` preserves the first workshop iteration.

`workshops/coding-with-ai/` is a draft scaffold for the next iteration.
