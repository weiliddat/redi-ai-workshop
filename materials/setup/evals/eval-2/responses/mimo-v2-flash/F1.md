### Step-by-Step Explanation

1.  **Understand the Data Structure**:
    The server sends a `timestamp` field in JSON format (usually an ISO string like `"2025-06-15T14:30:00Z"`). We need to capture this from `event.data` and convert it into a readable format (like `HH:MM:SS`).

2.  **Modify `addMessage` Function**:
    Currently, the `addMessage` function only accepts a single string. We need to update it to accept the raw message object so we can extract the timestamp directly from the source, rather than formatting it inside the event listener.
    *   **Why?** This keeps the formatting logic consistent for all message types (messages, joins, leaves, errors).

3.  **Format the Timestamp**:
    Inside `addMessage`, we will check if the data has a timestamp.
    *   If yes, create a `Date` object from the string.
    *   Use `.toLocaleTimeString()` to convert it to a local time string (e.g., "2:30:00 PM").
    *   **Why?** `toLocaleTimeString()` automatically handles the user's local time zone and browser locale settings.

4.  **Update the Message Display**:
    We will change the `textContent` to include the formatted time in square brackets (e.g., `[2:30:00 PM] Alice: Hello`).
    *   If no timestamp exists (for example, an error message we generate locally), we can hide the timestamp or use the current time as a fallback.

5.  **Update Event Listeners**:
    Modify the `socket.onmessage` handler to pass the parsed `data` object directly to `addMessage` instead of formatting the string first.

6.  **CSS Adjustments (Optional but Recommended)**:
    To make the timestamp stand out visually, we can style it with a lighter color and a fixed width.

    function addMessage(data) {
      const div = document.createElement('div');
      div.className = 'message';

      // 1. Determine the timestamp
      let timeString = '';
      if (data.timestamp) {
        try {
          // Create Date object and format to locale time string (e.g., "2:30:00 PM")
          const date = new Date(data.timestamp);
          timeString = date.toLocaleTimeString();
        } catch (e) {
          // Fallback if timestamp is invalid
          timeString = '';
        }
      }

      // 2. Construct the message text based on type
      let text = '';

      if (data.type === 'message') {
        // Format: [HH:MM:SS] Name: Text
        text = `[${timeString}] ${data.name}: ${data.text}`;
      } else if (data.type === 'join') {
        text = `[${timeString}] ${data.name} joined`;
      } else if (data.type === 'leave') {
        text = `[${timeString}] ${data.name} left`;
      } else if (data.type === 'nick') {
        // Handle nickname change
        text = `[${timeString}] ${data.oldName} is now known as ${data.name}`;
      } else if (data.type === 'list') {
        // Handle user list (specifically for /list command)
        text = `[${timeString}] Users online: ${data.names.join(', ')}`;
      } else if (data.type === 'error') {
        // Errors might not always have a timestamp from server, 
        // but if they do, we use it. Otherwise, we might skip the timestamp.
        if (timeString) {
          text = `[${timeString}] [ERROR] ${data.message}`;
        } else {
          text = `[ERROR] ${data.message}`;
        }
      }

      // 3. Display the text
      div.textContent = text;
      messagesEl.appendChild(div);
      messagesEl.scrollTop = messagesEl.scrollHeight;
    }
