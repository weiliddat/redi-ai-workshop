Your chat wouldn’t actually “chat” anymore — it would only send.

Nothing crashes, but you lose all incoming communication. Specifically:

- You won’t see any messages from other users  
- You won’t see your *own* messages echoed back (unless you manually add them client-side)  
- Join/leave notifications disappear  
- Server error messages won’t show  
- The UI will look “connected but silent,” which can feel like it’s broken  

Everything else still works:

- You can connect/disconnect  
- You can send messages to the server  
- The server will still process and broadcast messages  

So the app becomes write-only from the client’s perspective.
