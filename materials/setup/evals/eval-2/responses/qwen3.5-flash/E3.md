You can't send a JavaScript object directly through a WebSocket because WebSockets are a network protocol that only transmits raw data—text or binary—not native programming language objects like JavaScript's `{}`.

Here's why `JSON.stringify()` is necessary:

1. **WebSockets Send Bytes or Strings**  
   WebSockets work at the transport level, meaning they only send text or binary data—not JavaScript objects. To send structured data, it must be converted into a string first.

2. **Serialization Converts Objects to Transmissible Strings**  
   `JSON.stringify()` converts a JavaScript object into a JSON-formatted string suitable for network transmission. Here's an example:

   ```javascript
   // JavaScript object (cannot be sent directly)
   const message = { type: 'message', text: 'Hello' };

   // Convert to string before sending
   socket.send(JSON.stringify(message));

   // On the receiving side, parse the string back to an object
   const data = JSON.parse(event.data);
   ```

3. **Parsing on the Receiving End**  
   Since the client received a string, it must be parsed back into an object using `JSON.parse()` before it can be used meaningfully in JavaScript.

4. **Standard Data Format for Interoperability**  
   JSON is a universally supported, lightweight format that's easy to read, understand, and transmit. While you could use other formats (like `msgpack`), JSON is the most common due to its simplicity and built-in support in most web environments.

5. **Security and Compatibility**  
   Transmitting native objects isn't possible across protocol layers. Using standard text formats ensures compatibility across browsers, devices, and backends.

### Summary

You need `JSON.stringify()` because:
- WebSockets can't send JavaScript objects directly.
- Data must be serialized into a string for transmission.
- The receiving side must be able to decode the string back into usable data.

Without `JSON.stringify()`, your message would be sent as the literal string `"[object Object]"` instead of the structured JSON you intend to transmit.
